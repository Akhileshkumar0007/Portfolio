# Akhilesh Kumar Portfolio

Professional static portfolio website for a full stack developer.

## Tech

- HTML5
- CSS3
- JavaScript

## Run locally

Open `index.html` in a browser, or use a simple local server in your editor.

## Deploy to GitHub Pages

1. Create a GitHub repository and upload this project.
2. Keep `index.html` in the root of the repository.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/ (root)` folder.
6. Save and wait for GitHub Pages to publish the site.
